# Multiplayer Rock Paper Scissors

Multiplayer Rock Paper Scissors

This game was designed to allow for two individual human users to simultaneously player Rock Paper Scissors against one another in real time.

CSS3

Javascript ES6

Bootstrap 4.2.1 CSS
